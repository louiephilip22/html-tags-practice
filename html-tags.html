HTML Tags Cheat Sheet

Headings

Headings from h1 to h6 identify blocks and sections. Every page must have one h1.

Paragraphs

Paragraphs of text goes in p tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in ul tags. Each list item goes in an li tag.

Types of lists:

  • Unordered lists (ul)
  • Ordered lists (ol)

Ordered Lists

Ordered lists are numbered and goes in ol tags. Each list items goes in an li tag.

Anchors (i.e. links)

Anchor tags create links to othr pages. The URL goes in the "href" attribute. Go to Javascript.info to search for more information.

Settings 'target="blank"' will open the link in a new window. See MDN for more details.

Anchor tags can also create link to a new location on the same page. Instead of putting a URL, it goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allows you to Jump to the top of the page

Text Emphasis

Italics

The "em" element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

You can also use the "strong" element to emphasize text. This shows up as bold text in most browsers.
